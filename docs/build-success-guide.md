### Step-by-Step: Create Device Build in Appflow

1. **Open** Appflow dashboard: https://dashboard.ionicframework.com/apps/e0d0cbc9/build/builds
2. **Click** "New Build" button  
3. **Platform**: iOS
4. **Build Type**: Change from "simulator" to "development" 
5. **Signing**: Choose "Automatic" (let Appflow handle certificates)
6. **Click** "Build"
7. **Wait** ~3-5 minutes for build to complete
8. **Download** the IPA file

### Install IPA on Windows (No Mac needed!)

**Option A: AltStore (Free & Easy)**
1. Download AltStore: https://altstore.io/
2. Install on your Windows PC
3. Connect iPhone via USB
4. Install AltStore app on iPhone
5. Use AltStore to install your ScanEats IPA

**Option B: Sideloadly (Free)**
1. Download Sideloadly: https://sideloadly.io/
2. Connect iPhone via USB  
3. Drag your IPA into Sideloadly
4. Enter Apple ID and install

**Option C: 3uTools (Free)**
1. Download 3uTools
2. Connect iPhone
3. Go to Apps section
4. Install your IPA directly
